# Mi Portfolio Personal

Página web personal desarrollada con HTML, CSS y JavaScript. Portfolio profesional de desarrollador con diseño moderno, responsivo y modo oscuro/claro.

## 🚀 Características

- **Diseño responsivo**: Se adapta a móviles, tablets y escritorio
- **Modo oscuro/claro**: Toggle para cambiar entre temas
- **Secciones**: Inicio, Proyectos, Habilidades, Contacto
- **Formulario de contacto**: Con validación de campos
- **Navegación suave**: Smooth scroll entre secciones
- **Iconos**: Font Awesome para iconos profesionales

## 🛠️ Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Flexbox, Grid, Variables CSS
- **JavaScript** - Vanilla JS (sin dependencias)
